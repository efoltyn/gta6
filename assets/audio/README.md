# Cell Block Z audio assets

All runtime audio is loaded locally by `src/systems/audio.js`. The game no
longer generates audible effects from WebAudio oscillators or noise buffers.

## Sources

- `firearms/`: selected, trimmed, mono AAC versions of prepared CC0 recordings
  from [FreeFirearmsSFXLibrary](https://github.com/buddingmonkey/FreeFirearmsSFXLibrary).
  The original CC0 license is included in `firearms/LICENSE-CC0.txt`.
- `kenney/`: selected OGG files from Kenney's CC0
  [Impact Sounds](https://kenney.nl/assets/impact-sounds),
  [Interface Sounds](https://kenney.nl/assets/interface-sounds), and
  [RPG Audio](https://kenney.nl/assets/rpg-audio) packs. The original license
  files are included beside the audio.
- `oga/sfx100v2_*`: the CC0
  [100 CC0 SFX #2](https://opengameart.org/content/100-cc0-sfx-2) pack by
  rubberduck.
- `oga/wind-loop.ogg`: CC0
  [wind whoosh loop](https://opengameart.org/content/wind-whoosh-loop).
- `oga/car-engine-mid.m4a`: compact AAC conversion of a CC0
  [racing car engine sound loops](https://opengameart.org/content/racing-car-engine-sound-loops).
- `oga/city-loop.mp3`: CC0
  [City Loop](https://opengameart.org/content/city-loop-0).
- `oga/dark-place.ogg`: CC0
  [Dark Place (loop)](https://opengameart.org/content/dark-place-loop).
- `oga/siren.mp3`: CC0
  [Sirens and Alarm Noise](https://opengameart.org/content/sirens-and-alarm-noise).
- `oga/storm-siren.ogg`: CC0
  [Storm & Siren](https://opengameart.org/content/storm-siren).
- `rse/`: selected per-file CC0 sounds from
  [rse/soundfx](https://github.com/rse/soundfx). The upstream metadata sidecar
  for every selected file is stored beside its MP3.

Only CC0 files from `rse/soundfx` were selected.
